什么是H5？

H5即HTML5，是指“HTML”的第5个版本，而“HTML”，则是指描述网页的标准语言。因此，HTML5，是第5个版本的“描述网页的标准语言”。

H5中的一些新特性

  1. 用于绘画的 canvas 元素
  2. 用于媒介回放的 video 和 audio 元素
  3. 对本地离线储存的更好支持
  4. 新的特殊内容元素，比如 article、footer、header、nav、section
  5. 新的表单控件，比如 calendar、date、time、email、url、search

H5语义元素

  1. article 定义页面独立的内容qu
  2. aside 定义侧边栏内容
  3. bdi 允许你设置一段文本，使其脱离父元素的文本方向设置
  4. command 定义命令按钮，比如单选按钮、复选框按钮
  5. details 用于描述文档或文档某个部分的细节
  6. dialog 定义对话框，比如提示框
  7. summary 标签包含details元素的标题
  8. figure 规定独立的流内容（图片、图表、代码等）
  9. figcaption
  10. footer 定义 section 或 document 的页脚
  11. header 定义头部的文档
  12. mark 定义带有记号的文本
  13. meter 定义度量衡
  14. nav 定义导航链接的部分
  15. progress 定义任何类型的任务的进度
  16. ruby 定义ruby注释
  17. rt 定义字符的解释或发音
  18. rp 在ruby 注释中使用，定义不支持 ruby 元素的浏览器所显示的内容
  19. section 定义文档中的节
  20. time 定义日期或时间
  21. wbr 规定在文本中的何处添加换行符

H5 浏览器支持

最新版本的 Safari、Chrome、Firefox 以及 Opera 支持某些 HTML5 特性。Internet Explorer 9 将支持某些 HTML5 特性。

H5代码规范

  1. HTML5元素名可以使用大写和小写字母
  2. HTML5属性值不可以引用
  3. 图片通常使用alt 属性，
  4. 等号前后可以使用空格
  5. 使用HTML 编辑器 ，左右滚动代码是不方便的
  6. 样式表使用简洁的语法格式
